1. JavaScript 简介

JavaScript 是什么？

JavaScript 是一种让网页“活”起来的脚本语言。它可以响应点击、修改页面内容、和服务器通信，也可以处理用户交互。

JavaScript 在哪里运行？

主要在浏览器里，也可以在服务器上运行，比如 Node.js。

JavaScript 能做什么？

  • 改变网页内容
  • 响应用户操作
  • 与服务器交换数据
  • 保存少量客户端数据
  • 处理用户上传的文件
